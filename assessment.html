Multimídia Web

O elemento audio permite como elemento-filho o elemento source. Quais são os atributos que o source permite e qual conteúdo eles suportam? Quais os outros atributos do elemento audio e quando devem ser utilizados?

src: endereço do arquivo de mídia, type: o tipo do arquivo

controls: para poder controlar o audio, played: indicando que o audio foi todo reproduzido, src: a URL do audio a ser usado.

O elemento video permite como elemento-filho o elemento source. Quais são os atributos que o source permite e qual conteúdo eles suportam? Quais os outros atributos do elemento video e quando devem ser utilizados?

src: endereço do arquivo de mídia, type: o tipo do arquivo

controls: para poder controlar o audio, played: indicando que o audio foi todo reproduzido, height: altura da area de exibição do video, width: largura da area de exibição do video, src: a URL do video a ser usado.

No caso do canvas, HTML 5 nunca, em momento algum, fez qualquer animação. Isso porquê o HTML 5 é uma simples linguagem de marcação. Dessa forma, o que é necessário para criar animações com HTML5? O que são APIs do HTML5?

Para ser feita as animações com o uso do canvas precisamos usar uma linguagem de programação para desenhar dentro dessa tag, no caso o JavaScript.

As APIs do HTML5 são maneiras de atualizar os dados dos pixels em elemento HTML de forma programática, podendo construir cenas 2d e 3d. Exemplos são o próprio canvas e WebGL.

Recursos avançados

Faça uma breve descrição dos elementos/propriedades utilizadas e quais navegadores não são compatíveis com eles.

Scroll-behavior - serve para deixar a rolagem quando clica em algum link mais suave - Browsers não suportados: IE(todos), Edge(12-18), chrome(4 - 60) e safari(3.1 - 15.3)

FlexBox - serve para facilitar o posicionamento de elementos de forma flexível - Browsers não suportados: IE(6 - 9), Opera(10 - 11.5) e Opera Mobile(12)

Flex Gap - serve para criar espaçamento entre os elementos dentro de um container flexbox - Browsers não suportados: IE(todos), Edge(12-83), Chrome(4-83) e Safari(3.1-14)

Media Querys - serve para consulta com base na presença do apontador do usuário - Browsers não suportados: IE(todos), Firefox(2-63), Chrome(4-40) e Safari(3.1-8)

Grid - serve para montar a grade de layout em linhas e colunas - Browsers não suportados: IE(6-9), firefox(2-51), Chrome(4-56) e Safari(3.1-10)
